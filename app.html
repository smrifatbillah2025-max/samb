About This App

Smart Attendance Manager is a modern, user-friendly application designed to simplify attendance tracking and management for educational institutions.

এই অ্যাপ্লিকেশনটি শিক্ষা প্রতিষ্ঠানের জন্য উপস্থিতি ট্র্যাকিং এবং ব্যবস্থাপনা সহজ করার জন্য ডিজাইন করা হয়েছে।

Key Features

  • Create and manage multiple attendance sheets
  • Track student attendance with ease
  • Generate detailed mark sheets and reports
  • Print-ready attendance records
  • Dark mode support for comfortable viewing
  • Responsive design for all devices
  • Secure data storage with Firebase
  • Beautiful green-themed modern UI
